# My c++ cmake template

### Compiling
##### Windows
Generating build files
```powershell
mkdir build
cd build
cmake ..
```
Building for debug
```powershell 
cmake --build . --config Debug
```
Building for release
```powershell 
cmake --build . --config Release
```

##### Linux
Generating build files
```bash
mkdir build
cd build
cmake ..
```
Building for debug
```bash
cmake --build . --config Debug
```
Building for release
```bash
cmake --build . --config Release
```
